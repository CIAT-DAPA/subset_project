Methodologies

For the multivariate analysis, we have used three clustering algorithms:

1. DBSCAN - Density-Based Spatial Clustering of Applications with Noise: This technique is used to find arbitrary shaped clusters and to detect outliers (i.e. clusters with noise). DBSCAN requires two key parameters:

  • eps: Specifies how close points should be to each other to be considered neighbors and a part of a cluster. If the distance between two points is lower or equal to eps, these two points are considered neighbors.

  • minPts: This parameter specifies the minimum number of data points to form a dense region and, thus, a cluster..

Points that are in low-density regions are marked as outliers.

2. HDBSCAN - Hierarchical Density-Based Spatial Clustering of Applications with Noise : This algorithm extends DBSCAN by converting it into a hierarchical clustering algorithm. The parameter that should be defined for our implementation is min_cluster_size , which specifies the minimum number of points that a cluster should have. By default, this parameter is set to 10.

3. HAC - Hierarchical Agglomerative Clustering : HAC uses a bottom-up approach to form clusters. It starts by considering each point as a cluster, and merges pairs of clusters as moving up the hierarchy. The algorithm keeps merging until reaching the desired number of clusters set by the user via the parameter n_clusters. By default, this parameter is set to 5.
