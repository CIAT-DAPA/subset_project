This website offers to users many ways to search data about accessions, it has integrated many databases and linked this data to accessions in order to improve the information level. Though this tool offers many options to users to explore data, it does not provide a way to create subsets of accessions based on customizable indicators.

Subsets of accessions are a set of accessions of one or many Genebanks, which have special features that are requested by specific users, for example: a user may want to find a set of drought tolerance bean accessions, in this case, he/she should define what drought tolerance means based on a set of criteria, then search the accessions that match these criteria and finally create a list of these accessions.

This project focuses on designing, developing and testing a tool that allows users to create subsets of accessions through a web system using different data sources. To that purpose, the system requires to build a set of indicators to allow users to build novel queries based on data available in Genesys and custom datasets.


The Crop Trust is the donor of this project
