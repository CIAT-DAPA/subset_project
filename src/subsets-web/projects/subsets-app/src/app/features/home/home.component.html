Genebank collection subsetting tool

This website offers to users many ways to search data about accessions, it has integrated many databases and linked this data to accessions in order to improve the information level. Though this tool offers many options to users to explore data, it does not provide a way to create subsets of accessions based on customizable indicators.

Methodologies

DBSCAN

Density-Based Spatial Clustering of Applications with Noise.

HDBSCAN

Hierarchical Density-Based Spatial Clustering of Applications with Noise.

HAC

Hierarchical Agglomerative Clustering
